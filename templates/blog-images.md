# Blog Post Images

Header images and inline illustrations for technical blog posts.

## Hero Image — Code Concept

```
imgforge generate "An abstract representation of source code transforming into something visual. Dark background with floating lines of colorful code on the left side that dissolve into geometric particles on the right side. The particles form a subtle shape. Cinematic lighting with blue and purple tones. No text." -s landscape -q high
```

## Comparison Split

```
imgforge generate "A before/after split image. Left side labeled 'Before' shows a cluttered, complex code editor with many open tabs and messy code. Right side labeled 'After' shows a clean terminal with a single command and clean output. A vertical divider line separates the two halves. Flat illustration style." -s landscape -q medium
```

## Step-by-Step Illustration

```
imgforge generate "Three panels arranged horizontally showing a workflow: Panel 1 shows a text prompt being typed, Panel 2 shows gears processing, Panel 3 shows a finished image appearing. Each panel is numbered 1, 2, 3 in circles. Connected by arrows. Flat vector illustration style, blue and white color scheme." -s landscape -q medium
```

## Abstract Tech Pattern

```
imgforge generate "An abstract geometric pattern suitable as a blog post header. Interconnected nodes and lines forming a network graph on a dark gradient background. Nodes glow softly in teal and white. The pattern fades to transparent at the edges. Minimal, modern, tech-aesthetic." -s landscape -q medium
```
