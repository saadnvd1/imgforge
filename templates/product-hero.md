# Product Hero Shots

Clean product renders for landing pages, app stores, and social media.

## Mac App Icon

```
imgforge generate "A macOS app icon on a solid dark background. The icon is a rounded square with a subtle gradient from deep indigo to electric blue. A minimal white geometric symbol in the center — clean, modern, Apple-style. Soft ambient shadow beneath the icon. No text." -s square -q high
```

## CLI Tool in Terminal

```
imgforge generate "A dark terminal window showing a command-line tool in action. Green and white monospace text on a pure black background. The terminal has a macOS-style title bar with red/yellow/green dots. The output shows structured, colorful text with progress indicators. Realistic screenshot aesthetic, not illustration." -s landscape -q high
```

## App Screenshot with Device Frame

```
imgforge generate "An iPhone 15 Pro showing a minimal productivity app. The screen displays a clean list interface with soft shadows, rounded cards, and a bottom tab bar. The phone floats on a light gray gradient background with a subtle reflection beneath it. Photorealistic render." -s portrait -q high
```

## SaaS Dashboard

Realistic UI with proper spacing, typography hierarchy, and data visualization. No placeholder text — use realistic numbers." -s landscape -q high
