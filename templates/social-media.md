# Social Media Assets

Ready-to-post images for developer and tech audiences.

## Open Graph Image

```
imgforge generate "A minimal OG image for a blog post about building CLI tools. Dark navy background. Large bold white text that reads exactly: \"Building CLI Tools in Python\". Subtle code snippet decoration in the bottom-right corner, faded at 20% opacity. Clean, modern, developer-focused." -s landscape -q high
```

## GitHub Social Preview

```
imgforge generate "A GitHub repository social preview image. Dark gradient background from charcoal to near-black. A terminal window in the center showing a colorful CLI output. Below the terminal, small text in a monospace font. Minimal, developer-aesthetic, no corporate feel." -s landscape -q medium
```

## YouTube Thumbnail (Dev Content)

```
imgforge generate "A YouTube thumbnail for a programming tutorial. Split composition: left side has a close-up of code in a dark editor with syntax highlighting, right side has a terminal with green output text. Bold yellow text overlay that reads exactly: \"Zero Dependencies\". High contrast, attention-grabbing, no face." -s landscape -q high
```
